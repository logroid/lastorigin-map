# ラストオリジン 地域別 マップ(日本語版)
[https://lastoriginmap.github.io/](https://lastoriginmap.github.io/) の一部を日本語版のラストオリジンのデータに置き換えて公開しています。  
イベントを除き、日本語版で実装された敵の名前、スキル情報は全て置き換え済みです。  
イベントの敵に関しては順次対応していきます。
